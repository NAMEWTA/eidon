编码检测四阶段

fs::read(&path)                     // raw bytes
  ↓
sniff_bom(bytes)                    // 0xEFBBBF → UTF-8, 0xFFFE → UTF-16LE...
  ↓ (无 BOM)
chardetng::EncodingDetector         // 统计推断 GBK/Big5/Shift_JIS...
  ↓
encoding.decode_without_bom...      // 严格解码
  ↓ (失败)
encoding.decode()                   // 宽松解码
  ↓ (仍乱码)
String::from_utf8_lossy()           // 容错回退，永不报错
  → FileReadResult { content, encoding, language, had_bom }
