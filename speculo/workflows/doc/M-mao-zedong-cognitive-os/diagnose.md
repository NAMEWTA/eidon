# Diagnose Phase — 诊断分析

## 输入

- `speculo/.speculo/doc/<change>/problem-statement.md`（Phase 1 产物）
- Module A 的 8 大分析模型（见下方）

## 产物

- `speculo/.speculo/doc/<change>/analysis.md`

## 认知主线

**事实 → 矛盾 → 主次 → 定性 → 两面 → 条件 → 本质 → 具体 → 检验。**

## Module A · 8 大分析模型

### A1 · 矛盾分析法（主算法）
来源：《矛盾论》(017)

一句话：找到主要矛盾，就找到破局之道。

四个子工具：
1. **普遍性 vs 特殊性**："这个矛盾特殊在哪？"
2. **主要矛盾 vs 次要矛盾**："哪个一解决，其余迎刃而解？"
3. **矛盾的主要方面 vs 次要方面**："现在哪一方在支配？据此定性"
4. **同一性与斗争性**："逆转需要哪些条件？现在具备几个？"

原话："捉住了这个主要矛盾，一切问题就迎刃而解了。"
局限：信息不足时易误判主要矛盾。

### A2 · 实践认识论
来源：《实践论》(016)

一句话：认识从实践来，经理论提升，回到实践验证，循环往复、螺旋上升。
原话："你要知道梨子的滋味，你就得变革梨子，亲口吃一吃。"
局限：试错成本极高的领域（医疗、航天）需先充分理论推演。

### A3 · 调查研究方法论
来源：《反对本本主义》(006)、《〈农村调查〉的序言和跋》(058)

一句话：没有调查，没有发言权。
工序：选典型 → 选真正有经验的人 → 定纲目 → 开调查会 → 眼睛向下、口问手写。
局限：调查也得讲方法，"不做正确的调查同样没有发言权"。

### A4 · 实事求是
来源：《改造我们的学习》(059)、《整顿党的作风》(063)

一句话：一切从客观事实出发，从事物内部规律引出结论，"有的放矢"。
检查清单：判断从事实还是从愿望/本本出发？是否详细占有材料？结论是否从内部联系引出？

### A5 · 一分为二（两点论 + 重点论）
来源：《矛盾论》(017)；《关于正确处理人民内部矛盾的问题》(216)

一句话：任何事物都有两面，但两点之中又有重点。
动作：拿到任何判断先反问"它的反面是什么？"——这是避免片面化的安全阀。

### A6 · 阶级 / 结构分析
来源：《中国社会各阶级的分析》(000)、《怎样分析农村阶级》(008)

一句话：把局面拆成力量结构——分清敌/友/中间。
动作：分层 → 判定各方态度 → 找最广同盟 → 盯住摇摆者 → 团结多数、孤立主敌。
现代应用须中性化为"利益相关方/力量结构分析"。

### A7 · 透过现象看本质
来源：《星星之火，可以燎原》(005)

一句话："看它的实质，而把它的现象只看作入门的向导，一进了门就要抓住它的实质。"
判断真假强弱不看表面力量，看是否得人心、是否代表进步方向。

### A8 · 具体问题具体分析
来源：《中国革命战争的战略问题》(012)、《矛盾论》(017)

一句话：不同质的矛盾，用不同质的方法解决，反对一刀切照搬。
套用任何通用模型前先问三问：现在是哪个阶段？这个对象特殊在哪？不能照搬什么？

## 填写引导

### 1. 选择适用模型

从 Phase 1 确定的 `problem_type` 出发，从 8 模型中选用至少 2 个：

- 所有问题必过 A1（矛盾分析），这是主算法
- 信息来自二手而非亲历 → 必过 A3（调查研究）
- 涉及多方利益 → 必过 A6（结构分析）
- 判断涉及趋势预测 → 必过 A5（一分为二看转化）+ A7（看本质）
- 涉及"别人都这么做/标准做法" → 必过 A8（具体分析）

### 2. 逐一应用模型

每个选用的模型产出：

- **诊断结论**（1-2 句，带"不是……而是……"切割）
- **条件分析**（转化需要什么条件？现在具备哪些？）
- **局限标注**（这个判断的边界在哪？）

### 3. 综合定性

汇总各模型结论，回答三个问题：

1. **主要矛盾**是什么？为什么它是主要的？
2. **矛盾的主要方面**在哪一方？这一定性意味着什么？
3. **转化的关键条件**是什么？现在缺什么？

## 边界

- 不在此阶段给战略建议——先看清，再定打法。
- 每个判断必须带条件，"只有……才……"。
- 对无法确定的判断明确标注"待调查"。

## 完成准则

- 已应用至少 2 个 Module A 模型
- 每个模型结论带条件与局限
- 主要矛盾已明确写出且带论证
- `analysis.md` 无残留 `[TODO:]`
